LEA kuro kainų paieška ir Pricer.lt WordPress integracija.

== Installation ==

1. Įkelkite katalogą į `wp-content/plugins/kuras-pricer`.
2. Aktyvuokite įskiepį.
3. Nustatymuose įveskite API ir žemėlapio tiekėjo adresus.
4. Puslapyje įterpkite bloką „Kuro kainų palyginimas“ arba `[kuras_pricer]`.
